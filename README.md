# AI Query Chatbot

Lets people who don't know backend development spin up and manage a real database (MySQL or MongoDB) through:
1. **AI bot** - natural language ("make a students table with name, email, age") → real schema/queries, generated by Gemini and executed live.
2. **Query** - the raw SQL / mongo-shell view, editable and runnable directly.
3. **Table** - an Excel-like grid to view and edit rows directly.

All three stay in sync because they operate on the same underlying database.

## Structure
```
backend/    Express API (auth, AI, MySQL/Mongo execution)
frontend/   React + Vite + Tailwind UI
docker-compose.yml   Local MySQL + MongoDB for testing
```
